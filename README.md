# 🎬 Movie Browser App

A React Native app built with Expo for browsing trending movies.

## 🚀 Features
- Browse trending movies
- View movie details
- Add to favorites
- Responsive UI
- Navigation with React Navigation

## 🛠️ Tech Stack
- React Native + Expo
- React Navigation
- Axios for API requests (https://www.themoviedb.org/)
- AsyncStorage for favorites
